## FASM Mini Programs

This repository contains a collection of small programs written in FASM for a course by xorpd.

## Requirements

To build the programs, you need to install FASM. You can download it from the official website: https://flatassembler.net/download.php .

## Building

After installing FASM, assemble a source file with:

```bash
fasm <filename>.asm
```

This will generate an executable file.

## Running

To run the generated executable, use:

```bash
<filename>.exe
```
